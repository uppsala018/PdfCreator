# Composer Font Assets

This folder contains fonts intentionally embedded by the professional composer.

Allowed sources:

- Google Fonts fonts with an open-source license, usually SIL Open Font License.
- Open Foundry can be used as a discovery source for open fonts.
- Fontesk fonts only when the specific font page states `Free for commercial use, OFL`.
- 1001 Fonts can be used through its `Free Fonts for Commercial Use` index,
  but each downloaded font still needs an individual license check.

Do not add:

- Personal-use-only fonts.
- Proprietary fonts.
- Fonts with unclear redistribution terms.
- Fontesk downloads that say only `Free for personal use`.

Current bundled fonts:

- Cormorant Garamond Regular/Bold from Google Fonts.
- EB Garamond Regular/Bold from Google Fonts.
- Libre Baskerville Regular/Bold from Google Fonts.
- Lato Regular/Bold from Google Fonts.
- Montserrat Regular/Bold from Google Fonts.
- Poppins Regular/Bold from Google Fonts.

Fontesk, Open Foundry, and 1001 Fonts can be used as discovery libraries for
future premium-looking typefaces, but each candidate must be checked at the
individual font page or upstream repository before adding the font file to this
repository.
